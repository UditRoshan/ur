Wrote my first HTML code in NOTEPAD. Had no idea what i was doing then. Just did it for the sake of marks in Computer Science in school. But still got 60% marks in Computer.

Started learning JAVA as a backbencher in my class. I know it is quite a jump from HTML to JAVA. But that is what our school management decided. Now back to JAVA. This year I paid no attention at all. Didn't know what was a 'Class' in JAVA. I don't know how i passed in Computer Science.

Still was continuing with JAVA. Paid a bit attention, but not much. Maybe knew about 10% of what was being taught in the class.

There it is. The turning point. It was Class 10th. Paid a lot attention. Tried to grasp but I was actually getting nothing. Gradually started practicing as the exam approached. And actually managed to score 80% in Computer Science. Yeah!

Now I realised that I have missed a great opportuinity. But there was no time to regret. As I started my class 11th, I decided to complete my Front End Web Developement. Yes it was overwhelming at that time. But I was fascinated. So revised my HTML, which I learnt in class 6th. Knew a lot new things in it. And finally switched from NOTEPAD to VISUAL STUDIO CODE.

Knew HTML, it was time to move to CSS. I thought it would be very difficult as I was going to learn a whole new language. But to be honest it was not that difficult. Knew all the properties and started making projects. Made a lot of them and got a strong grip on CSS. But I am pretty sure I don't know 100% of it. Apart from coding, I started Graphic Designing as well. Got my hand on Photoshop, Illustrator and InDesign. Made a lot of stuffs. Learnt a lot.

Now I had the frame(HTML), parts and accesories(CSS). It was time to know the how the engine works. Started JAVASCRIPT. And Oh boy! It was difficult to me. I don't know why. Maybe I was approaching it in a difficult way. But i was like - what am I doing?

Slowly got used to it. Now I was able to understand concepts. Then I started including JS in my code my website. And after some horrendous errors, I am slowly getting it. Not 100%. But I now can say I know more than what is needed to make a website interactive. Now currently I am in my process of working with ReactJs. And also did some Php. Yes going to start Back End Developement and Android Developement soon.
